
USB(uart)Command: 

Enable Raw data output
```
d;  
```
Disable Raw data output
```
D;  
```
Switch to PA mode:
```
l;   
```
Switch to Probe(endstop) mode:
```
e;
```
set thrhold:xx; xx must be number, recommend number is 3 to 10. for example, we want to set the value to 8
```
8;  
```
note: if you use other com port software, remember to add the end character "\n"
